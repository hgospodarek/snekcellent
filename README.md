## Snekcellent

- Rails 5
- Postgres
- Bootstrap

### Goals:
- have a side project
- track snake care and behavior because snakes aren't cats
- try out Rails 5
- try out a front end framework that's not Foundation

### Long term:

- add React or Vue to this


## Do Stuff
1. Clone `git@github.com:hgospodarek/snekcellent.git`
2. `cd snekcellent`
3. `rake db:create`
4. `rake db:migrate`
